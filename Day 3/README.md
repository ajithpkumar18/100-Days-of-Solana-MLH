# Reusable Solana Wallet (Node.js)

This project demonstrates how to get balance and convert sol to lamports and vice versa.

---

## 🚀 Features

- Understand SOL and Lamports

---

## 🧪 Usage

1. Replace address variable value with any public key
2. Get the balance in lamports and sol

You only need to fund the wallet **once** for devnet usage.

---

## 📌 Notes

- Compatible with Solana tooling
- Ideal for learning and local development
- Avoid using this pattern in production

---

## 🛠 Tech Stack

- Node.js
- Web Crypto API
- Solana Web3 / Kit utilities
